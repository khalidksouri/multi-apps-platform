# Math4Child v4.0.0

Application éducative n°1 pour apprendre les maths en famille !

## 🚀 Caractéristiques

- ✅ Interface moderne et responsive
- ✅ 12 langues supportées
- ✅ Plans d'abonnement compétitifs
- ✅ TypeScript strict
- ✅ Next.js 14.2.30
- ✅ Tailwind CSS

## 📦 Installation

```bash
npm install
```

## 🛠️ Développement

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 🌐 Déploiement

Ce projet est optimisé pour Netlify avec export statique.

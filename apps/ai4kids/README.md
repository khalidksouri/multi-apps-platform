> **Application hybride moderne - Web + Android + iOS**

## 🚀 Démarrage Rapide

```bash
# Installation
npm install

# Développement
npm run dev
# Ouvrir http://localhost:3000

# Build web
npm run build

# Build mobile
npm run build:capacitor
```

## 📱 Déploiement Mobile

```bash
# Android
npm run cap:add:android
npm run deploy:android

# iOS (macOS uniquement)
npm run cap:add:ios
npm run deploy:ios
```

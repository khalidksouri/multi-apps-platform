## 🚀 **DÉMARRAGE RAPIDE**

### **Installation et Développement**
```bash
# Installation des dépendances
npm install

# Démarrage en mode développement
npm run dev

# Tests complets
npm run test
npm run test:ui
npm run test:stress
npm run test:api

# Build pour production
npm run build

# Applications mobiles
npm run build:capacitor
npm run android:build
npm run ios:build
```
